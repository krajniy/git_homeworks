First level: Укажите:

что такое ветка в терминологии GIT;

		Изолированный поток разработки, другая версия проекта.

команды, с помощью которых можно создать ветку и переключаться между ветками.
		
		git branch somebranch - создать ветку somebranch
		git branch (-v) - посмотреть список веток
		git checkout somebranch - переключиться на новую ветку
		git checkout master - вернуться в мастер-ветку
